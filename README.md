# UKTA – Grade 9 Science Quiz

Live quiz app for Nepal CDC Grade 9 Science & Technology SEE exam prep.

## How to host on GitHub Pages (FREE, takes 3 minutes)

### Step 1 — Create a GitHub account
Go to https://github.com and sign up (free)

### Step 2 — Create a new repository
- Click the **+** button → **New repository**
- Name it: `ukta-quiz` (or anything)
- Set it to **Public**
- Click **Create repository**

### Step 3 — Upload the file
- Click **Add file** → **Upload files**
- Drag and drop `index.html` into the box
- Click **Commit changes**

### Step 4 — Enable GitHub Pages
- Go to **Settings** → **Pages** (left sidebar)
- Under **Source**, select **Deploy from a branch**
- Branch: **main**, Folder: **/ (root)**
- Click **Save**

### Step 5 — Done! 🎉
After ~1 minute your site will be live at:
`https://YOUR-USERNAME.github.io/ukta-quiz`

Share this link with anyone — works in every browser!
